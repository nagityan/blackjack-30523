## 説明

カードゲームのブラックジャックで遊べるサイトです。
ユーザーとしてログインすれば、より多くのポイントが付与されます。また、他プレーヤーたちとポイントを競えます。



## 使用技術
- Ruby 2.6.5
- Ruby on Rails 6.0.0
- Mysql 5.6.47
- Nginx
- AWS
  - EC2
  - Capistrano 3.14.1
- Cicle CI 2.1
- Rspec
